Introduction

01/12/2016

Quartz is a material research project that unites state of the art computer-aided rapid prototyping technology (CNC) with glass - one of the most versatile design materials for storage and packaging.

Using open source CNC hardware and custom-developed software, this speculative project aims to develop a novel computer-aided material process for the use of glass in the formation of two and three-dimensional artifacts. This project can be broadly divided into 3 categories:

  • Material Research
  • Machine Customization
  • Parametric modelling

You’ll need to be familiar with basic CAD modelling and CNC fabrication. The Resources page lists the breakdown of setup you’ll need in order to get started. It is highly recommended that you go through all the materials before practicing any experiments with glass. There is a bit of a learning curve involved in carrying out the extrusions.
